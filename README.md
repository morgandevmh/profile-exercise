# profile-exercise
exercise to create a Business Card, with some mandatory instructions in HTML and CSS

# EXERCISE (PROMO 3):

## Create a page named: 
profile.html  and its associated styles in  style.css . The page must display a profile card with the following elements:
- A photo (use a free image or an external link)
- The user's full name
- Their job or role
- A short description of 2-3 sentences
- A “Contact me” button (non-functional)

## HTML instructions
- Use structure tags: ``<!DOCTYPE html> ,  <html> ,  <head> ,  <body>``

  
- Structure the card with tags: ``<div> ,  <img> ,  <h1> ,  <h2> ,  <p> ,  <button>`` 


- Link the style sheet  style.css  with ``<link>`` 



## CSS instructions
- Center the card in the middle of the page
- Add a light background to the map and rounded corners
- Make the image circular and limit its size to 150px wide
- Style the titles, text, and button so that they are harmonious
- Add an effect when hovering over the button (background color changes)
- Add an icon or link to a social network under the description
- Use a shadow effect on the map
- Make the button interactive with a hover effect
- Use CSS variables as seen in class this morning

## Deliverables
- Files: profile.html and style.css
- In a public GitHub repository with README.md
- The code must be valid and readable
- The page must be visually appealing
